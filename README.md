# LexControl separado

## Arquivos
- `index.html`
- `styles.css`
- `config.js`
- `app.js`
- `vercel.json`
- `supabase-schema.sql`

## Antes de subir
1. Abra `config.js`
2. Troque:
   - `COLE_SUA_SUPABASE_URL_AQUI`
   - `COLE_SUA_SUPABASE_ANON_KEY_AQUI`

## Supabase
1. Rode `supabase-schema.sql` no SQL Editor
2. Crie o usuário em Authentication > Users

## Vercel
Suba esta pasta no GitHub e importe no Vercel.
